# Biscoito da Sorte

Plugin para KOReader que exibe frases inspiradoras no estilo biscoito da sorte. Desenvolvido com o auxílio do DeepSeek. Testado no Kindle Basic 2022.

## Funcionalidades

- Frases online via APIs gratuitas (sem autenticação)
- Fallback para frases locais quando offline
- Arquivo `frases.txt` personalizável
- Ativação/desativação de APIs individuais

## Instalação

Copie a pasta `biscoitodasorte.koplugin` para a pasta de plugins do KOReader e reinicie o dispositivo. O plugin aparecerá em **Ferramentas → Biscoito da Sorte**.

## Uso

- Toque em **Abrir biscoito** para ver uma frase.
- No diálogo, use **Novo** para outra frase, **Fechar** para sair.
- Em **Frases locais** pode restaurar o padrão, ver ou adicionar frases.

## Frases locais

O arquivo `frases.txt` (criado automaticamente) aceita uma frase por linha.  
Para incluir autor, separe com `|` ou tab: `Frase|Autor`.

## APIs

- ZenQuotes, Quotable, Advice Slip – ativadas/desativadas em **Configurar APIs**.

# Fortune Cookie

KOReader plugin that displays inspirational fortune‑cookie messages. Developed with the help of DeepSeek. Tested on Kindle Basic 2022.

## Features

- Fetches quotes from free online APIs (no authentication)
- Local fallback using a customizable `frases.txt` file
- Toggle individual APIs on/off

## Installation

Copy the `biscoitodasorte.koplugin` folder to the KOReader plugins folder and restart the device. The plugin will appear under **Tools → Fortune Cookie**.

## Usage

- Tap **Abrir biscoito** to see a quote.
- Use **Novo** for another, **Fechar** to close.
- Manage local phrases in the **Frases locais** submenu.

## Local phrases

The `frases.txt` file supports one quote per line. Optional author: `Quote|Author`.
